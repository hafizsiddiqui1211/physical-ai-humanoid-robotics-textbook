# Physical AI & Humanoid Robotics Textbook

A comprehensive textbook on Physical AI and Humanoid Robotics, implemented with Docusaurus for deployment on GitHub Pages.

## About

This textbook provides a complete guide to understanding and implementing advanced robotic systems that integrate artificial intelligence with physical embodiment. It covers essential concepts and practical implementations needed to understand and develop modern humanoid robots and physical AI systems.

## Features

- **12+ chapters** covering foundational concepts through advanced topics
- **50+ executable code examples** demonstrating robotics concepts
- **20+ diagrams** illustrating key concepts and architectures
- **Simulation-to-real transfer** methodologies and workflows
- **Integration with modern frameworks**: ROS 2, Isaac Sim, Gazebo, Unity
- **Vision-Language-Action (VLA)** systems and voice-to-action interfaces
- **Complete capstone project** integrating all concepts

## Content Structure

The textbook is organized into the following sections:

- **Foundations**: Core concepts in Physical AI and Embodied Intelligence
- **Systems**: Robotic systems architecture using ROS 2
- **Simulation**: Digital twin environments and simulation frameworks
- **AI & Perception**: Advanced perception and navigation systems
- **Humanoid Robotics**: Kinematics, dynamics, and locomotion control
- **Vision-Language-Action**: Multi-modal AI systems
- **Capstone Project**: Complete system integration and assessment

## Deployment

This textbook is built with [Docusaurus](https://docusaurus.io/) and deployed using GitHub Pages.

## Usage

The textbook is available at: https://hafizsiddiqui1211.github.io/physical-ai-humanoid-robotics-textbook/

## Development

To run the textbook locally:

1. Clone the repository
2. Navigate to the `docusaurus` directory
3. Install dependencies: `npm install`
4. Start development server: `npm start`
